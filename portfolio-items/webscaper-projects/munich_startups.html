Munich Startups Portal Webscraper and Dashboard - Personal Project

I built a webscraper to obtain data from the official Munich startups portal and created a dashboard in Tableau using this dataset.

Data Acquisition

In this project, I built a webscraper using Scrapy to obtain data about Munich startups. The resulting spider traversed the pages of the offical start-up portal for Munich, collecting as much information about the individual businesses as possible. Information that could be gathered included: addresses, contact details, business plan, growth stage, and the number of employees.

I created a dashboard on Tableau public using this dataset

Key tools & libraries

  • Python: scrapy, pandas, geopy
  • Tableau: actions, filters, calculated attributes
  • github
