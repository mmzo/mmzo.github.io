Google Data Analytics Certification - Capstone Project

Google Capstone project as the final assessment for the Google Data Analytics Specialisation on Coursera.

User Trends

Analysis

In this project I analysed data from a fictional Chicago-based bike rental company for 13 months. The project was for a data analytics team who are seeking to assist the marketing team in improving membership uptake. The task was primarily to explore the current data and uncovering differences between member and casual users.

I used Rmarkdown to document the data wrangling and analysis process and created a dashboard visualisation using Tableau. I wanted to understand the role of user type and bicycle selection preferences, so I included this as part of the dashboard. Finally, all data can be explored based on time using the month and year selectors.

Each station can be selected from the map, which filters the data displayed in the other parts of the dashboard. This involved creating global filters, and setting up a dashboard action that applied to the map itself.

Key Tools & Libraries:

  • R: dplyr, tidyverse
  • Rstudio: R-Markdown
  • Tableau
