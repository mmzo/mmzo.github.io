Analysis of Youtube Trending Videos in Germany - Personal Project

Jupyter notebook analysis of the Kaggle dataset: Trending Youtube Video Statistics.

Data Exploration

In this project I analysed 7 months of data from the Trending Youtube Video Statistics dataset. The goal was to gather insights about trending videos within Germany specifically and answer questions such as: What types of videos trended the most? How long did they trend for? Was there a relationship between the number of views and whether a video trends or not?

The full analysis is documented in a Jupyter notebook hosted on my Kaggle page.

Key tools & libraries:

  • Python: pandas, matplotlib, seaborn, plotly
  • Jupyter Notebooks
